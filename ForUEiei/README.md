# เว็บบอกรัก (Simple Password Reveal)

ไฟล์ที่สร้าง:
- `index.html` — หน้าเว็บหลัก
- `style.css` — สไตล์สวย ๆ
- `script.js` — เช็คพาสเวิร์ดและแสดงหัวใจ

วิธีใช้งาน:
1. เปิดไฟล์ `index.html` โดยดับเบิลคลิกหรือเปิดในเว็บเบราว์เซอร์
2. พิมพ์รหัส (ค่าเริ่มต้นคือ `รักนะ`) แล้วกด `ส่ง`

ถ้าต้องการเปลี่ยนรหัส ให้แก้ตัวแปร `PASSWORD` ใน `script.js` เป็นรหัสที่ต้องการ

ถ้าต้องการรันเซิร์ฟเวอร์ท้องถิ่น (ไม่จำเป็น แต่ช่วยบางฟีเจอร์) ให้ใช้คำสั่งในโฟลเดอร์นี้:
```bash
python -m http.server 8000
# แล้วเปิด http://localhost:8000
```
